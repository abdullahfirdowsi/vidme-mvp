# 🎥 VidMe — AI Video Pitch for Job Seekers

VidMe helps job seekers stand out by generating short, personalized AI video pitches for job applications and LinkedIn outreach.

## 🚀 Problem

Applying to jobs is broken:
- Resumes get ignored
- Recruiters spend <10 seconds per profile
- No way to stand out

## 💡 Solution

VidMe creates a 30-second AI-generated video where you introduce yourself — helping you get noticed faster.

## ⚙️ Features (MVP)

- AI-generated pitch scripts
- Talking avatar video generation
- Role-based personalization
- Shareable video links

## 🛠️ Tech Stack (Planned)

- LLM: Groq (Llama 3 / Mixtral)
- Video: Wav2Lip / LivePortrait
- Hosting: Azure (Student Credits)
- GPU: RunPod (optional)
